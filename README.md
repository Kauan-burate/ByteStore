# 💻 ByteStore

Projeto desenvolvido para a disciplina de Desenvolvimento Web, simulando uma loja virtual de informática moderna e responsiva.

## 👨‍💻 Integrantes

- Vitor Plaut
- Kauan Burate

## 🌍 Temática

Mundo Real

A ByteStore foi criada com o objetivo de simular uma loja virtual de informática, oferecendo aos usuários uma experiência próxima à encontrada em grandes e-commerces do mercado. O sistema permite visualizar produtos, pesquisar itens, filtrar categorias, acessar promoções e conhecer mais sobre a empresa.

O projeto busca aplicar na prática os conhecimentos de HTML, CSS, Bootstrap e JavaScript, desenvolvendo uma interface moderna, organizada e responsiva.

---

## 🎯 Objetivos do Projeto

- Desenvolver uma loja virtual de informática.
- Aplicar conceitos de desenvolvimento web.
- Utilizar componentes responsivos com Bootstrap.
- Implementar filtros e buscas dinâmicas.
- Trabalhar com manipulação de dados utilizando JavaScript.
- Simular funcionalidades reais de um e-commerce.
